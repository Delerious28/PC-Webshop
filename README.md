# PC-Webshop

Concept specification for a high-performance PC hardware webshop with a dark, gaming-inspired aesthetic, rich faceted filtering, and a compatibility-aware custom PC builder.

## Live prototype
- Open `index.html` directly in a browser or serve locally with `python -m http.server 8000` and visit http://localhost:8000.
- Includes faceted filters with active chips, skeleton loading, hover specs, cart fly-out, and a compatibility-aware PC builder with socket/RAM checks, wattage budgeting, and GPU clearance validation.
